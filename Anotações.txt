Comandos para executar
Com um MP3:
python main.py process sua_musica.mp3 --language pt --tempo 120
A conversão MP3 → WAV é automática. O pipeline inteiro em 1 comando.
Outros comandos:
python main.py process arquivo.ogg              # OGG
python main.py process arquivo.wav              # WAV
python main.py capture --duration 30            # gravar do sistema
python main.py full --duration 60               # gravar + processar
python main.py serve                            # API em http://localhost:8000
python main.py diagnose                         # diagnosticar
Testes:
python -m pytest tests/ -v -k "not slow"        # sem modelos ML (13 tests)
python -m pytest tests/ -v -m slow              # com modelos ML
